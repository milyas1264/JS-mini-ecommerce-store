# Mini Ecommerce Store

A simple JavaScript Ecommerce Store project built using HTML, CSS, and Vanilla JavaScript.
The application fetches products from an external API and allows users to search, filter, and add items to a cart.

## 📌 Features

* Display products from API
* Product details page
* Search products
* Category filter
* Add to cart
* Remove from cart
* Cart counter
* Cart total price calculation
* LocalStorage cart persistence
